# 说明书形式审查清单（S-1 ~ S-32）

> 审查范围：专利申请初稿中的"说明书"部分（技术领域/背景技术/发明内容/附图说明/具体实施方式）
> 数据来源：约32个知识产权保护中心预审自检表（2026-06-10全中心汇总）
> 目标保护中心：由用户选择，影响分歧项/独有项判定
> 注：本清单仅审说明书文本本身，不含请求书/委托书。

---

## A类规则（⚡ 脚本自动扫描）

| 编号 | 规则 | 提出中心 |
|------|------|----------|
| S-1 | 各部分发明/实用新型名称一致（含角标格式） | 统一要求 |
| S-2 | 发明名称字数：一般≤25字；化学领域≤40字（审查指南原文）；极个别中心≤60字（重庆/吉林/苏州/辽宁/福建/陕西，超指南） | 统一+中心差异 |
| S-3 | 说明书五部分完整（技术领域/背景技术/发明内容/附图说明/具体实施方式），各小标题前无段号/空行/空格 | 统一要求 |
| S-4 | 发明说明书不得出现"本实用新型"；实用新型不得出现"本发明" | 统一要求 |
| S-5 | 不应使用"如权利要求……所述的……"引用语 | 统一要求 |
| S-6 | 说明书正文不得含有插图（含图的表格也算插图） | 统一要求 |
| S-7 | 表格清晰完整（含表号/表题/表头/表身） | 统一要求 |
| S-8 | 不得含有"在此处键入""QUOTE""本地目录路径"等模板残留 | 合肥（独有，其余💡建议关注） |
| S-9 | 不得出现"其特征在于"用语 | 吉林、辽宁（部分中心明确禁止） |
| S-10 | 公式清晰无乱码，符号参数有明确释义 | 统一要求 |
| S-11 | 附图说明与附图一一对应（无缺图/缺说明/多余附图/相同附图） | 统一要求 |
| S-12 | 同组成部分附图标记一致；文字未提及标记不在附图出现，反之亦然 | 统一要求 |
| S-13 | 无多余标点/空行/下划线/波浪线 | 统一要求 |
