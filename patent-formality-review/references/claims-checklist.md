# 权利要求书形式审查清单（C-1 ~ C-30）

> 审查范围：专利申请初稿中的"权利要求书"部分
> 数据来源：约32个知识产权保护中心预审自检表（2026-06-10全中心汇总）
> 目标保护中心：由用户选择，影响分歧项/独有项判定
> 注：本清单仅审权利要求书文本本身，不含请求书栏位。

---

## A类规则（⚡ 脚本自动扫描）

| 编号 | 规则 | 提出中心 |
|------|------|----------|
| C-1 | 用阿拉伯数字顺序编号，序号不重复 | 统一要求 |
| C-2 | 每项权利要求只在结尾有一个句号，无多余标点，无"，。"连用 | 统一要求 |
| C-3 | 不得出现非择一引用及多项从属权要引用多项从属权要（多引多） | 统一要求 |
| C-4 | 从属权利要求只能引用在前的权利要求 | 统一要求 |
| C-5 | 不应出现"如说明书……所述""如图……所示"等引用说明书或附图语句 | 统一要求 |
| C-6 | 附图标记应加括号，置于对应技术特征后面，与说明书及附图一致 | 统一要求 |
| C-7 | 同一部件的附图标记前后一致 | 统一要求 |
| C-8 | 不得含有不确定用语："厚/薄/强/弱/高温/高压"等含义不确定词 | 山西、合肥、内蒙古、苏州、洛阳、甘肃、无锡、长春 |
| C-9 | 不得含有"例如/优选/尤其是/必要时/可能/通常/约/接近/等"等不确定表述 | 山西、合肥、内蒙古、苏州、洛阳、甘肃、无锡、长春 |
| C-10 | 不得含有导致不清楚的括号 | 山西、合肥 |
| C-11 | 不得存在"所述XXX"缺乏引用基础 | 统一要求 |
| C-12 | 不得含有插图（流程图/方框图/曲线图/相图等）；化学式/数学式须清晰列参数 | 统一要求 |
| C-13 | 公式符号有明确释义，XML格式公式清晰无乱码 | 统一要求 |
| C-14 | 专利类型用语：发明权要不出现"本实用新型"，实用新型不出现"本发明" | 统一要求 |
