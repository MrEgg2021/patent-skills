## 二、重构内容

### 2.1 全局规则重编号（解决编号冲突）

按5部分重排，每部分独立编号段，**互不重叠**：

| 部分 | 新编号段 | 旧编号 |
|---|---|---|
| 权利要求书 | **C-1 ~ C-NN** | F-9,14-46（混乱） |
| 说明书 | **S-1 ~ S-NN** | F-47-78（跳号） |
| 说明书附图 | **D-1 ~ D-NN** | F-79-101（跳号） |
| 说明书摘要 | **AB-1 ~ AB-NN** | F-1-16 |
| 摘要附图 | **AF-1 ~ AF-NN** | （原散在drawings/abstract） |

用字母前缀（C/S/D/AB/AF）替代纯F-数字，从根上杜绝跨部分撞号。scanner、4个checklist、报告模板同步用新编号。

> 内容以 `2026-06-10-formality-5parts-rules.md` 的 A~F 节为准逐条落编号。

### 2.2 中心×维度精确映射表（核心新增，解决"做不到按中心输出"）

在 scanner 中新建 `CENTER_RULES` 配置，把汇总文档的差异维度建成数据表：

```python
# 发明名称字数上限（说明书/摘要中的名称）—— 已联网核实审查指南原文
# 指南原文(第一部分第一章4.1.1 + 第二部分第二章2.2.1): 通则≤25字；化学领域某些发明可放宽到40字。
# "60字"非指南原文，属极个别中心(重庆/吉林/苏州/辽宁等)自检表自加口径。
NAME_LENGTH_DEFAULT = (25, 40)   # (通则上限, 化学领域上限) — 适用所有中心、最严格模式
NAME_RELAXED_60 = ["重庆","吉林","苏州","辽宁","福建","陕西"]  # 极个别放宽到60(超指南,默认不采纳)
# 判定: 默认按25/化学40; 选到放宽中心时可按其60字口径，报告标注"超出审查指南通则"
# 附图颜色（互斥维度）
FIGURE_COLOR = {
    "禁彩色":["内蒙古","苏州","常州","合肥","成都","四川","湖北"],
    "可彩色":["北京","吉林","济南"],
    "发明可/实用新型禁":["湘潭","长沙"],
}
# 摘要字数
ABSTRACT_LEN = {  # 多数300；青岛/天津特殊计字法
    "青岛":(300,"字母按1字符"), "天津":(300,"字母按1字符"),
    # 其余默认300
}
# 附图清晰度
FIGURE_CLARITY_2_3 = ["陕西","青岛","浙江","辽宁","大连","天津"]
```

判定逻辑：scanner 收到 target_center → 查这些表 → 取该中心的具体值判定。这是"按中心精确输出"的核心机制。

### 2.3 独有/互斥规则配置（替换现零散的 CENTER_DISPUTE_CONFIG）

按汇总文档，把每条差异规则建成 `{规则号: {type, 适用中心, 具体值}}`：
- **互斥型(mutual_exclusive)**：附图颜色、化学名称字数（同维度不同中心相反值）
- **独有型(unique)**：陕西"3-6-8-7"、成都/洛阳"中央领导人名字"、长春"人脸/二维码"、南昌"发表论文/人脸马赛克"、西安"计量单位/名称居中"、湖南"全部发明人/标准地图"、青岛"其特征是用语"等
- **数值型(quantitative)**：名称字数、摘要字数、清晰度

### 2.4 最严格模式修复（修实测确认的bug）

`get_center_verdict` 增加"是否最严格模式"判断：
- 未覆盖中心（最严格模式）：互斥维度取**最严值**（附图禁彩色、名称≤25字、摘要须有句号）；独有项**全部按🔴**（不再返回suggestion跳过）。
- 已覆盖但该项未要求：返回💡建议关注，且**查出来照样列入"建议关注区"**（不再 `return` 跳过不查）。
- 修改所有 `if verdict=="suggestion": return` → 改为"查出来标💡"。

### 2.5 只审5部分（剔除手续类）

按用户确认范围，从4个checklist和scanner中**剔除**：请求书、实质审查请求书、委托书、盖章/签字、备案、费减、发明人身份证、领域限制、非正常申请文号、不予通过阈值、注册地址、经济社会效益等。
**保留**：5部分内部/之间的一致性（术语、附图标记对应、摘要附图须是说明书附图之一、说明书名称与摘要一致）、发明名称（说明书/摘要里的）、序列表/保藏（说明书组成部分）。

### 2.6 中心列表更新

第0步的19中心列表 → 更新为实际约30中心。补充：广州、重庆、青岛、新乡、湘潭、洛阳、南昌、湖南、大连、无锡、成都、济南、四川、徐州、北京、安徽、湖北、吉林等（去重后全集）。同源中心标注差异（辽宁≈大连差名称字数；济南≈四川差附图颜色）。
