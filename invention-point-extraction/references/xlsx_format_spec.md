# xlsx 输出格式规范

## 列结构

| 列 | 列名 | 列宽 | 对齐 | 自动换行 |
|---|------|------|------|---------|
| A | 专利公开号 | 18 | 居中 | 否 |
| B | 专利标题 | 40 | 靠左 | 是 |
| C | 技术问题 | 60 | 靠左 | 是 |
| D | 技术方案 | 60 | 靠左 | 是 |

## 多发明点格式

同列内有多个发明点时，在单元格内部按以下规则书写：

```
1. [第一个问题描述]
2. [第二个问题描述]
3. [第三个问题描述]
```

- **序号格式**：`1. `（数字 + 英文句号 + 空格）
- **分隔方式**：序号项之间用 `\n`（换行符，非段落间距）分隔
- **无额外空行**：相邻序号项之间不要插入空行
- **与技术方案一一对应**：技术方案列中的 `1.` 对应技术问题列的 `1.`

## 单发明点场景

只有一个发明点时，不写序号 `1.`，直接写内容。多发明点才加序号。

## 表头样式

- 加粗
- 背景色：浅灰（#F2F2F2）
- 冻结首行

## 空值处理

- 无法提取专利公开号 → 填文件名（不含扩展名）
- 无法提取专利标题 → 填"（未识别）"
- 某篇专利说明书无法解析 → 该行保留，技术问题/方案填"解析失败：[原因]"
